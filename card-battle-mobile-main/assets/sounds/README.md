# 🔊 ملفات الصوت

ضع ملفات الصوت هنا بالأسماء التالية:

| الملف | الحدث |
|---|---|
| `attack.mp3` | ضربة الهجوم |
| `win.mp3` | فوز الجولة |
| `loss.mp3` | خسارة الجولة |
| `ability.mp3` | تفعيل قدرة |
| `next_round.mp3` | الانتقال للجولة التالية |
| `draw.mp3` | تعادل |

> جميع الأصوات يتم تحميلها مسبقاً (`Audio.Sound.createAsync`) عند بدء المعركة.
> إذا لم يوجد الملف، سيتجاهل النظام الخطأ بصمت.

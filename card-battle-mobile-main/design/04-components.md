# Components

| Component | Role |
|---|---|
| `CardTile` | Card preview in the selection grid |
| `DeckSlot` | Selected card slot in the deck area |
| `BattleCard` | Large combat card shown in the arena |
| `StatBadge` | ATK / DEF / HP display unit |
| `HPBar` | Animated health bar |
| `RoundCounter` | Current round info |
| `ScoreBoard` | Player vs bot live score |
| `ResultBanner` | Round or match outcome |
| `ActionButton` | Main action button |

# Feedback

Every action gets a response. Every outcome gets felt.

---

## Events

| Event | Visual | Haptic |
|---|---|---|
| Card selected | Scale up + border glow | Light |
| Card removed | Fade down | None |
| Round start | Cards slide in | None |
| Round win | Green flash + particles | Medium |
| Round loss | Red flash + shake | Heavy |
| Match win | Celebratory burst | Strong |
| Match loss | Dark fade | Soft |
| Critical HP | Red pulse on HP bar | None |

---

## Rules

- Haptic weight must match the emotional weight of the event.
- Never animate something the player didn’t cause.
- Win feels bigger than a round win.
- Loss must feel real, not ignored.

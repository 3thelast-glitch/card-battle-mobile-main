# Feedback & Motion

| Event | Response |
|---|---|
| Card selected | Scale up + border glow + light haptic |
| Card removed | Fade down |
| Round start | Cards slide in from opposite sides |
| Round win | Green flash + particle burst + medium haptic |
| Round loss | Red flash + shake + heavy haptic |
| Match win | Strong celebratory burst |
| Match loss | Dark overlay + quiet fade |
| Critical HP | Pulsing red bar |

## Rules
- Motion should feel fast and intentional
- Haptics must match the weight of the event
- Never use animation for decoration alone

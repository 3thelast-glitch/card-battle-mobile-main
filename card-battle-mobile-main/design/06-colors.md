# Colors

---

## Base Palette

| Token | Value | Use |
|---|---|---|
| `bg.base` | `#0f0f1a` | App background |
| `bg.surface` | `#1a1a2e` | Cards and panels |
| `bg.elevated` | `#16213e` | Modals and overlays |
| `text.primary` | `#ffffff` | Main text |
| `text.secondary` | `#a0a0a0` | Labels and hints |
| `accent.primary` | `#e94560` | Actions and highlights |
| `accent.win` | `#22c55e` | Win state |
| `accent.loss` | `#ef4444` | Loss state |
| `accent.draw` | `#f59e0b` | Draw state |
| `border.subtle` | `#1e293b` | Dividers and frames |

---

## Rules

- Win is always green. Loss is always red. Never swap them.
- Legendary gold must always stand out on dark surfaces.
- Accent colors are for meaning, not decoration.
